Meet App

This PWA hosted on Amazon Lambda fetches upcoming development-related events using the Google Calendar API. Login is handled by Google OAuth2. Once users login, they can filter the display of events by specifying their preferred city or how many events they want to see at once. I built this application using test-driven development (TDD) and Enzyme for code testing.
